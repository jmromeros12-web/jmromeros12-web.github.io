What Can JavaScript Do?

JavaScript can change HTML content.

Aqui hay un arbol

El DOM (Document Object Model) es la representación en forma de árbol de una página web, donde cada etiqueta HTML es un objeto en ese árbol. Las etiquetas son los bloques de construcción de HTML, como <html>, <head> y <body>, que se anidan para formar la estructura jerárquica del documento. El DOM es fundamental porque permite a lenguajes como JavaScript interactuar con la página, modificando su contenido, estructura y estilo.<>
